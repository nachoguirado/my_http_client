# My HTTP Client

An extensible, object-oriented HTTP client for Python, designed to simplify API integrations and promote clean architecture. Supports GET, POST, PUT, DELETE, and custom HTTP methods with configurable authentication, middleware for retries/logging/caching, and pluggable response parsers (JSON, HTML, XML). ## Features

- Support for multiple HTTP methods (GET, POST, PUT, DELETE...).
- Dynamic headers and query parameters.
- Middleware for retries, logging, and caching.
- Parsers for JSON, HTML, and other formats.
- Object-Oriented design, easy to extend.
